Overview

An experimental project to help students better visualize 3D graphs (it takes some getting used to!). By mapping hand movements to movements which explore the graph, we hoped to make the abstract graph more physical and, for want of a better word, tangible. I didn't make the holographic display (I had great teammates who did that part), but the software, and interfacing with the Leap Motion hand tracker, was all done by me.

( please excuse the movember beard )

Main Features

  • Works with any 3D graph function
  • Hand-based controls for rotation and panning
  • Secret 'Star Wars' mode to throw Palpatine

Technology Used

  • Unity3D
  • Leap Motion
  • Pepper's Ghost Holographic Effect (I didn't do this part though)
