Overview

The Learning Management System used by my university is problematic in 2 main ways: itâ€™s difficult to download course materials (too many clicks to get to the download pages, and you can only download individual files), and it looks very old (modern websites tend to favour cleaner designs). I was frustrated by these two pain points, so I made a chrome extension that allows for one-click recursive downloads of course materials, and makes the LMS look (objectively, in my opinion) better. More than 25% of the undergraduate population is currently using my unofficial extension.

Main Features

  • One-click button to recursively download course materials
  • Stylistic improvements (looks way better!)
  • No change to the underlying LMS - everything is done on top of it on the user's browser

Technology Used

  • JavaScript (ES6) (for the collated download functionality)
  • Chrome Extensions API
  • Sass (to override the ugly default styling)
